Help

Insert key,value :

Insert a new (key,value) pair. Key is the song title and value is a string

Search key :

Search for a specific key and print its value. If key = *, it returns every (key,value) pair that has been saved in DHT from each node

Delete key :

Deletes a (key,value) pair specified by key.

Overlay :

Prints the topological order of the network. The nodes in the chord ring are printed with order in which they are connected.

Depart :

Node departure. The node leaves the system gracefully.
